Microsoft Fabric – Training on Microsoft Learn(Thilaga)
         Get started with Microsoft Fabric is a training course available on Microsoft Learn that introduces learners to the basics of Microsoft Fabric. It helps users understand how to work with data analytics, data engineering, and business intelligence tools in one platform. The course provides step-by-step learning modules and hands-on exercises to build practical skills. It is useful for beginners who want to learn modern data analytics technologies.
Cybersecurity Fundamentals – Training on Microsoft Learn
         Get started with Cybersecurity Fundamentals is a training course available on Microsoft Learn that introduces learners to the basics of protecting digital information and systems. It helps students understand common threats, security practices, and tools used to safeguard data in today’s connected world. The course provides step‑by‑step modules and practical exercises to build awareness and technical skills. It is especially useful for beginners who want to learn how to stay safe online and explore careers in digital security.
